CYOA Reincarnation Project

Overview: We applied what we learned, Markdown and Command Line, To collaborate with our groupmates to create a "Choose Your Own Adventure" game.

The purpose of this project was to practice collaboration while creating a project with a topic we liked. This project was to be done by using our understanding of Markdown and Command Line.

This project was neither hard or easy. At first it wasn't that hard but over time, me and my groupmates found problems in code and encountered merge conflicts. The way we resolved the problems were fixing them together. My group tried to work together to find out what was wrong with the code and in the end multiple perspectives helped out a lot.

After the end result, There are many improvements to be made. Some things I would have done differently would be the design. I would add pictures so it didn't seem so bland and more options for paths.
